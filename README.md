** Proyecto Base de Datos **
📌 Descripción

Este repositorio contiene el diseño, modelado e implementación de una base de datos relacional desarrollada con MySQL, administrada mediante DBeaver y modelada visualmente con Draw.io.

El proyecto incluye:

Modelo entidad-relación 

Modelo lógico

Normalización

Script de creación de base de datos

Scripts de inserción de datos

** 🛠 Tecnologías Utilizadas **

🐬 MySQL – Sistema de gestión de base de datos relacional.

🗄 DBeaver – Herramienta para administración y ejecución de consultas SQL.

🎨 Draw.io – Herramienta para el diseño del modelo entidad-relación.
